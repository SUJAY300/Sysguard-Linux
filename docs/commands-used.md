# Linux Commands Used

| Command | Purpose |
|----------|----------|
| hostname | Machine hostname |
| whoami | Current user |
| uname | Kernel information |
| date | Current date |
| uptime | Uptime |
| free | Memory statistics |
| df | Disk usage |
| ps | Running processes |
| ip | Network interfaces |
| ping | Connectivity |
| grep | Filtering |
| awk | Parsing |
| head | Top entries |
| cut | Field extraction |
| sort | Sorting |
| wc | Counting |
| tee | Logging |
| mkdir | Directory creation |
| chmod | Permissions |
| printf | Formatted output |
| source | Load configuration |
